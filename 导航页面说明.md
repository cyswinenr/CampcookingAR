## 💻 技术实现

### 1. NavigationActivity.kt

**主要功能：**
```kotlin
class NavigationActivity : AppCompatActivity() {
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        
        // 获取传递的团队信息
        val teamName = intent.getStringExtra("teamName") ?: "野炊小组"
        binding.teamNameText.text = teamName
        
        setupListeners()
    }
    
    private fun setupListeners() {
        // 微课视频区域点击
        binding.videoSection.setOnClickListener {
            Toast.makeText(this, "进入微课视频", Toast.LENGTH_SHORT).show()
            // TODO: 跳转到微课视频页面
        }
        
        // 过程记录区域点击
        binding.recordSection.setOnClickListener {
            Toast.makeText(this, "进入过程记录", Toast.LENGTH_SHORT).show()
            // TODO: 跳转到过程记录页面
        }
    }
}
```

### 2. 布局文件

**横屏布局 (layout-land/activity_navigation.xml)：**
- 使用 `ConstraintLayout` 作为根布局
- 顶部 `LinearLayout` 显示团队信息
- 底部 `LinearLayout` 横向排列两个 `CardView`
- 左右各占50%（weight=1）

**竖屏布局 (layout/activity_navigation.xml)：**
- 使用 `ConstraintLayout` 作为根布局
- 顶部 `LinearLayout` 显示团队信息
- 底部 `LinearLayout` 纵向排列两个 `CardView`
- 上下各占50%（weight=1）

### 4. 跳转逻辑

**从 MainActivity 跳转：**
```kotlin
private fun startCooking() {
    // 显示欢迎信息
    Toast.makeText(this, "欢迎 ${teamInfo.getTeamName()}！", Toast.LENGTH_SHORT).show()
    
    // 跳转到导航页面
    val intent = Intent(this, NavigationActivity::class.java)
    intent.putExtra("teamName", teamInfo.getTeamName())
    startActivity(intent)
    overridePendingTransition(android.R.anim.fade_in, android.R.anim.fade_out)
    finish()
}
```
