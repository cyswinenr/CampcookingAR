## 🎯 设计要点

### 1. **大数字显示**
```xml

    android:textSize="32sp"      ← 从16sp提升到32sp（2倍）
    android:textStyle="bold"     ← 加粗字体
    android:text="0/3" />
```

### 2. **状态指示器**

| 状态 | 图标 | 文字 | 颜色 |
|------|------|------|------|
| 未开始 | 📌 | 未开始 | 灰色（subtitle_color） |
| 进行中 | 📌 | 还需X张/段 | 橙色（fire_orange） |
| 已达标 | ✅ | 已达标 | 绿色（nature_green） |

## 🔧 技术实现

### RecordActivity.kt 核心代码

```kotlin
private fun updateProgressUI() {
    val photoCount = ...
    val videoCount = ...

    // 更新大数字
    binding.photoProgressText.text = "${photoCount}/3"
    binding.videoProgressText.text = "${videoCount}/1"

    // 三种状态逻辑
    when {
        photoCount >= 3 -> {
            // 已达标：绿色
            binding.photoProgressStatusText.text = "✅ 已达标"
            binding.photoProgressText.setTextColor(GREEN)
        }
        photoCount == 0 -> {
            // 未开始：灰色
            binding.photoProgressStatusText.text = "📌 未开始"
            binding.photoProgressText.setTextColor(BLUE)
        }
        else -> {
            // 进行中：橙色
            val remaining = 3 - photoCount
            binding.photoProgressStatusText.text = "📌 还需${remaining}张"
            binding.photoProgressText.setTextColor(ORANGE)
        }
    }
}
```
