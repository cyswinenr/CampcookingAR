# 📱 学生端过程记录 - 快速使用指南

## 🎯 什么是过程记录？

过程记录是帮助你记录野炊每个环节、进行自我评价的工具。通过拍照、评分、选标签的方式，让你回顾和总结自己的表现。

---

## 🚀 快速开始

### 1. 进入记录页面

```
封面页 → 团队信息登记 → 导航页 → 点击"过程记录" → 开始记录
```

### 2. 五个流程阶段

| 阶段 | 图标 | 要做什么 |
|------|------|----------|
| 准备阶段 | 📋 | 检查食材和工具，做好分工 |
| 生火阶段 | 🔥 | 搭建灶台，点燃柴火 |
| 煮饭阶段 | 🍚 | 淘米煮饭，控制火候 |
| 炒菜阶段 | 🥘 | 清洗切配，炒制菜品 |
| 完成阶段 | ✅ | 用餐收拾，整理场地 |

---

## 📸 操作步骤

### 每完成一个阶段，做3件事：

#### ① 拍照记录（推荐3-5张）
```
点击 [📸 拍照] 按钮
↓
拍摄过程照片
↓
照片自动保存
```

**拍什么？**
- 准备阶段：食材、工具
- 生火阶段：柴火摆放、火势
- 煮饭阶段：淘米、煮饭过程、米饭成品
- 炒菜阶段：食材处理、烹饪过程、成品
- 完成阶段：用餐、清理场地

#### ② 自我评分（1-5星）
```
拖动评分条 ⭐⭐⭐⭐⭐
```

| 星级 | 评价 | 说明 |
|------|------|------|
| ⭐⭐⭐⭐⭐ | 非常好 | 我们做得很棒！|
| ⭐⭐⭐⭐ | 很好 | 表现不错！|
| ⭐⭐⭐ | 还行 | 还可以，继续努力 |
| ⭐⭐ | 需努力 | 下次要更认真 |
| ⭐ | 待改进 | 需要多练习 |

#### ③ 选择标签（可多选）

**👍 做得好的地方**（绿色标签）
- 点击选中
- 可以选多个

**💪 需要改进的地方**（红色标签）
- 诚实记录问题
- 下次改进

#### ④ 完成并进入下一阶段
```
点击 [✓ 完成本环节] → 点击 [下一步 →]
```

## 💡 温馨提示

### ⏰ 自动计时
- 系统会自动记录每个阶段的用时
- 顶部显示总用时
- 无需手动计时

### 💾 自动保存
- 所有记录自动保存
- 可以随时退出
- 重新打开会恢复进度

### 📷 照片管理
- 点击照片可查看大图
- 长按可删除照片
- 建议每个阶段2-5张照片

### 🔄 阶段切换
- 可以点击左侧列表切换阶段
- 建议按顺序完成
- 已完成的阶段显示 ✓

## ❓ 常见问题

### Q1: 必须拍照吗？
**A:** 不强制，但强烈建议！照片是最好的记录方式。

### Q2: 评分标准是什么？
**A:** 根据自己的真实感受评分，没有标准答案。主要看：
- 完成质量
- 团队配合
- 操作规范
- 效率高低

### Q3: 能修改之前阶段的记录吗？
**A:** 可以！点击左侧的阶段卡片，可以回到之前的阶段修改。

### Q4: 忘记记录了怎么办？
**A:** 可以事后补充，点击相应阶段补充照片和评价。

### Q5: 不小心退出了怎么办？
**A:** 不用担心，所有记录都自动保存了，重新进入会恢复。

## 🎯 评价小技巧

### 1. **客观真实**
- 如实记录表现
- 不夸大也不贬低
- 诚实面对问题

### 2. **发现亮点**
- 多关注做得好的地方
- 记录团队配合
- 肯定每个人的贡献

### 3. **反思改进**
- 思考哪里可以做得更好
- 记录遇到的困难
- 计划下次怎么改进

### 4. **及时记录**
- 每完成一个阶段立即记录
- 趁记忆清晰时评价
- 不要等到最后一起记

## 🌟 评价示例

### 优秀示例：

- 评分：⭐⭐⭐⭐

- 问题标签：水放多了

**反思总结**：
这个例子很好地记录了过程，既肯定了优点，也诚实指出了小问题。
