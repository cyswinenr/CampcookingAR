# 快速使用指南

## 🚀 三步启动服务器

### 第一步：安装Python（如果还没有）

1. 访问 https://www.python.org/downloads/
2. 下载并安装Python 3.7或更高版本
3. 安装时勾选"Add Python to PATH"

### 第二步：安装依赖

打开命令行（CMD或PowerShell），进入服务器目录：

```bash
cd O:\PadSoftware\CampcookingAR\teacherserver
pip install -r requirements.txt
```

### 第三步：启动服务器

**Windows用户：**
双击 `start_server.bat` 文件

**或者使用命令行：**
```bash
python app.py
```

## 📱 配置学生端和教师端

### 学生端配置

1. 打开学生端APP
2. 完成所有数据录入
3. 点击"提交给教师"
4. 输入服务器显示的IP地址（例如：`192.168.1.100`）
5. 点击提交

### 教师端配置

1. 打开教师端APP
2. 输入服务器IP地址（与服务器启动时显示的IP一致）
3. 点击"刷新"获取学生数据
4. 查看学生数据并进行评价

## 🔍 常见问题

### Q1: 服务器启动后显示什么？

服务器启动后会显示：
```
============================================================
🏕️  野炊教学数据管理系统 - 教师端服务器
============================================================
服务器地址: http://192.168.1.100:5000
Web管理界面: http://192.168.1.100:5000/
API状态查询: http://192.168.1.100:5000/api/status
============================================================
```

**重要：** 记住显示的IP地址，学生端和教师端都需要使用这个IP。

### Q2: 如何查看学生数据？

有两种方式：

1. **使用教师端APP**（推荐）
   - 输入服务器IP地址
   - 查看学生列表和详细数据
   - 进行评价

2. **使用Web界面**
   - 在浏览器中访问：`http://服务器IP:5000/`
   - 查看基本信息

### Q3: 数据存储在哪里？

所有数据存储在 `data/` 目录下：
- `data/students/` - 学生数据
- `data/evaluations/` - 教师评价
- `data/exports/` - 导出的ZIP文件

### Q4: 如何备份数据？

直接复制 `data/` 文件夹即可。

### Q5: 端口被占用怎么办？

编辑 `config.py` 文件，修改 `PORT` 的值：
```python
PORT = 5001  # 改为其他端口
```

### Q6: 防火墙阻止连接怎么办？

**Windows:**
1. 打开"Windows Defender 防火墙"
2. 点击"允许应用通过防火墙"
3. 找到Python，勾选"专用"和"公用"

**或者临时关闭防火墙进行测试**

### Q7: 如何停止服务器？

在服务器运行的命令行窗口中，按 `Ctrl+C`

## 💡 使用技巧

1. **保持服务器运行**
   - 在学生提交数据期间，保持服务器运行
   - 不要关闭命令行窗口

2. **定期备份**
   - 课程结束后，备份 `data/` 文件夹
   - 或使用导出功能导出所有数据

3. **网络检查**
   - 确保笔记本和学生端/教师端在同一WiFi网络
   - 如果无法连接，检查IP地址是否正确

4. **查看日志**
   - 服务器会在命令行显示日志信息
   - 如有错误，查看错误信息进行排查
