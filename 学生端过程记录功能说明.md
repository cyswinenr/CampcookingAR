# 学生端过程记录功能说明

## 📚 功能概述

学生端过程记录系统是一个**以教育为导向**的自我评价工具，帮助学生在野炊过程中：
- ✅ 记录每个环节的过程
- ✅ 进行自我反思和评价
- ✅ 培养自主学习能力
- ✅ 积累活动经验

---

## 🎯 教育设计理念

### 1. **流程化指导**
按照野炊自然流程设计5个阶段：
1. 📋 **准备阶段** - 检查食材和工具
2. 🔥 **生火阶段** - 搭建灶台并点燃柴火
3. 🍚 **煮饭阶段** - 淘米煮饭
4. 🥘 **炒菜阶段** - 清洗切配并炒制菜品
5. ✅ **完成阶段** - 用餐和收拾

每个阶段都有：
- **温馨提示**：操作要点提醒
- **自动计时**：培养时间管理意识
- **照片记录**：留存过程痕迹
- **自我评价**：反思表现

### 2. **正向激励导向**
- ⭐ 5星评价系统（而非扣分制）
- 👍 优点标签（鼓励发现亮点）
- 💪 改进标签（理性看待问题）
- 🎉 完成庆祝（增强成就感）

### 3. **简化操作**
- 📷 拍照为主（减少文字输入负担）
- 🌟 点选评分（快速完成自评）
- 🏷️ 标签选择（标准化描述）
- ⏰ 自动计时（无需手动记录）

---

## 💡 核心功能

### 1️⃣ 流程节点管理

**界面左侧**显示所有阶段列表：
```
○ 未开始  ▶ 进行中  ✓ 已完成
```

**功能**：
- 当前阶段高亮显示（黄色背景）
- 显示每个阶段的用时
- 完成的阶段显示评分
- 点击可切换阶段（带确认）

### 2️⃣ 拍照记录

**大按钮设计**：`📸 拍照`
- 点击立即调用相机
- 照片自动保存到当前阶段
- 网格展示，可删除

**教育价值**：
- 培养记录习惯
- 提供真实证据
- 便于后续回顾
- 增强参与感

### 3️⃣ 自我评价系统

#### ⭐ 星级评分（1-5星）
- 5星：非常好 - "我们做得很棒！"
- 4星：很好 - "表现不错！"
- 3星：还行 - "还可以，继续努力"
- 2星：需努力 - "下次要更认真"
- 1星：待改进 - "需要多练习"

#### 🏷️ 标签系统

**以"生火阶段"为例：**

**优点标签**（绿色）：
- 速度很快
- 柴火摆放好
- 通风良好
- 安全操作
- 火势稳定

**问题标签**（红色）：
- 多次点火
- 柴火潮湿
- 烟雾太大
- 火势不稳

**设计考量**：
- 优点标签多于问题标签（7:3比例）
- 语言正面积极
- 问题标签中性描述（非批评性）

### 4️⃣ 自动计时功能

**全程计时**：
- 顶部显示总用时
- 每个阶段独立计时
- 自动记录开始/结束时间

**教育意义**：
- 培养时间观念
- 了解各环节耗时
- 提高效率意识

### 5️⃣ 数据持久化

**自动保存**：
- 切换阶段时保存
- 点击评分/标签时保存
- 暂停/退出时保存

**恢复功能**：
- 重新打开App自动恢复
- 继续上次进度
- 所有数据保留

## 📊 数据结构

### ProcessRecord（过程记录）
```kotlin
{
  teamInfo: 团队信息,
  startTime: 开始时间,
  endTime: 结束时间,
  currentStage: 当前阶段,
  stages: {
    PREPARATION: StageRecord,
    FIRE_MAKING: StageRecord,
    ...
  }
}
```

### StageRecord（阶段记录）
```kotlin
{
  stage: 阶段类型,
  startTime: 开始时间,
  endTime: 结束时间,
  photos: [照片路径列表],
  selfRating: 自我评分(1-5),
  selectedTags: [选中的标签],
  isCompleted: 是否完成
}
```

## 🎯 使用流程

### 学生操作步骤：

1. **进入记录页**
   - 从导航页点击"过程记录"
   - 系统自动开始第一阶段（准备阶段）

2. **完成一个阶段**
   ```
   拍照记录 → 自我评分 → 选择标签 → 点击"完成本环节"
   ```

3. **进入下一阶段**
   - 点击"下一步"按钮
   - 系统自动切换并开始计时

4. **全部完成**
   - 完成最后一个阶段
   - 弹出庆祝提示
   - 可生成报告

### 时间估算：
- 每个阶段评价：**约1-2分钟**
- 全程评价总时长：**约5-10分钟**
- 拍照不计入评价时间

## ✨ 教育价值

### 1. **培养自我评价能力**
- 学会客观评价自己的表现
- 识别优点和不足
- 建立改进意识

### 2. **提高过程意识**
- 关注"过程"而非仅"结果"
- 理解每个环节的重要性
- 培养细致观察能力

### 3. **增强团队协作**
- 记录中涉及团队表现
- 反思团队配合
- 促进相互学习

### 4. **建立时间观念**
- 了解各环节所需时间
- 学会合理安排
- 提高效率意识

### 5. **积累实践经验**
- 照片+评价形成完整记录
- 便于回顾总结
- 为下次活动提供参考
