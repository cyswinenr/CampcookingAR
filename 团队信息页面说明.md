# 📋 团队信息采集页面说明

## ✅ 已完成的功能

### 页面概述

创建了第二个页面（MainActivity），用于采集野炊小组的基本信息，包括：
1. 学校名称
2. 年级
3. 班级
4. 炉号
5. 小组人数
6. 人员姓名

## 📝 输入字段详解

### 1. 学校 (School)
- **图标：** 📍 地图图标
- **类型：** 文本输入
- **提示：** "学校名称"
- **验证：** 必填，不能为空

### 2. 年级 (Grade)
- **图标：** 📅 日历图标
- **类型：** 文本输入
- **提示：** "年级（如：九）"
- **验证：** 必填，不能为空
- **示例：** 九、八、七

### 3. 班级 (Class)
- **图标：** 📋 视图图标
- **类型：** 文本输入
- **提示：** "班级（如：1班）"
- **验证：** 必填，不能为空
- **示例：** 1班、2班、3班

### 4. 炉号 (Stove Number)
- **图标：** 🧭 指南针图标
- **类型：** 文本输入
- **提示：** "炉号（如：A1）"
- **验证：** 必填，不能为空
- **示例：** A1、B2、C3
- **颜色：** 🔴 红色图标（区分左右）

### 5. 小组人数 (Member Count)
- **图标：** 📝 日程图标
- **类型：** 数字输入
- **提示：** "小组人数"
- **后缀：** "人"
- **验证：** 必填，必须大于0
- **颜色：** 🔴 红色图标

### 6. 人员姓名 (Member Names)
- **图标：** ✏️ 编辑图标
- **类型：** 多行文本输入
- **提示：** "人员姓名"
- **帮助：** "每行一个姓名，或用逗号分隔"
- **最小行数：** 3行
- **最大行数：** 5行（横屏）
- **验证：** 必填，不能为空
- **颜色：** 🔴 红色图标

## 💾 数据模型

### TeamInfo 类

```kotlin
data class TeamInfo(
    var school: String = "",           // 学校
    var grade: String = "",            // 年级
    var className: String = "",        // 班级
    var stoveNumber: String = "",      // 炉号
    var memberCount: Int = 0,          // 小组人数
    var memberNames: String = ""       // 人员姓名
)
```

**方法：**
- `isValid()` - 验证信息是否完整
- `getTeamName()` - 获取团队完整名称

## 💬 对话框

### 1. 确认信息对话框

**触发：** 点击"开始野炊"按钮且信息验证通过

**内容：**
```
学校：XX中学
年级：九
班级：1班
炉号：A1
人数：6人
成员：张三、李四、王五...

确认信息无误？
```

**按钮：**
- 确认 → 保存信息，开始野炊
- 修改 → 返回继续编辑

### 2. 重置确认对话框

**触发：** 点击"重置"按钮

**内容：**
```
确定要清空所有输入的信息吗？
```

**按钮：**
- 确定 → 清空所有输入
- 取消 → 不做任何操作

### 3. 退出确认对话框

**触发：** 按返回键

**内容：**
```
确定要退出应用吗？
```

**按钮：**
- 确定 → 退出应用
- 取消 → 留在当前页面
